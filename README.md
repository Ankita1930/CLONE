# Spotify Clone (UI Design)

This project is a Spotify-inspired user interface built using HTML and CSS.  
It focuses on recreating the layout and design of the Spotify web app, including sidebar navigation and music cards.

## Features
- Spotify-like dark theme UI  
- Sidebar navigation (Home, Search, Library)  
- Search bar design  
- Music cards (Hindi songs, Top 50, Global, etc.)  
- Grid layout for playlists  
- Clean and responsive design  

## Note
This project is a frontend UI clone only.  
Music playback functionality is not implemented.

## Technologies Used
- HTML  
- CSS
